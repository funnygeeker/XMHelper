## 事件函数
?> 2.0.0-beta.2

### 基本事件

#### 接口事件
函数名：`self.on_echo()`
##### 说明
接收到事件类型为 “API接口返回” 的事件时返回为 `True`。若附加参数，则需要事件为 “API接口返回” 且 echo 字段与参数相等才返回 `True`
##### 参数
| 字段名 | 数据类型 | 默认值 | 说明 |
| :---- | :---- | :---- | :---- |
| `echo` | Any | `None` | echo内容（多为 str 或 int） |
##### 返回
| 数据类型 | 可能的值 | 说明 |
| :---- | :---- | :---- |
| bool | `True`、`False` | 无 |

##### 说明

| `keyword` | str | - | 关键词 |
##### 返回
| 数据类型 | 可能的值 | 说明 |
| :---- | :---- | :---- |
| bool | `True`、`False` | 无 |


#### 关键词匹配
函数名：`self.on_keywords_match()`
##### 说明

##### 参数
| 字段名 | 数据类型 | 默认值 | 说明 |
| :---- | :---- | :---- | :---- |
| `keywords` | str、list[str] | - | 关键词 |
##### 返回
| 数据类型 | 可能的值 | 说明 |
| :---- | :---- | :---- |
| bool | `True`、`False` | 无 |

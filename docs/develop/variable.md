### 数据

#### 原始数据
| 变量名 | 数据类型 | 可能的值 | 说明 |
| :---- | :---- | :---- | :---- |
| `self.data` | dict | - | 原始数据 |

#### 数据通道
##### 说明
您可以将变量代入此字典，用于向之后的插件传输数据
##### 示例
`self.channel['这是数据'] = '这是数据'`

| 变量名 | 数据类型 | 可能的值 | 说明 |
| :---- | :---- | :---- | :---- |
| `self.channel` | dict | - | 数据通道（用于插件之间传递变量） |
